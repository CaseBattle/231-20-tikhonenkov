# Portfolio Auth (PHP + SQLite)

Простое веб‑приложение с регистрацией и входом по номеру телефона и паролю. Страницы реализованы на PHP, данные хранятся в локальной базе `data/database.db`. Запуск осуществляется через Node.js, который поднимает встроенный PHP-сервер.

## Возможности

- Регистрация с проверкой уникальности номера телефона  
- Авторизация по номеру и паролю, сессии PHP  
- Восстановление пароля по номеру  
- Перенаправление на страницу личного портфолио после входа  
- Выход с главной страницы с возвратом на форму логина

## Стек

- PHP 8+ (PDO + SQLite)  
- SQLite (`data/database.db`)  
- Node.js 18+ для запуска PHP-сервера

## Подготовка

```bash
git clone <repo>
cd portfolio-auth-php
npm install
```

При первом запуске PHP автоматически создаст базу и таблицу пользователей.

## Запуск

```bash
npm start
```

По умолчанию сервер доступен по адресу <http://127.0.0.1:8080>.  
Переменные окружения:

- `HOST` — адрес (по умолчанию `127.0.0.1`)  
- `PORT` — порт (по умолчанию `8080`)  
- `PHP_PATH` — путь к бинарю `php`, если он не находится в `PATH`

## Структура

```
includes/          вспомогательные PHP-файлы (PDO, сессии)
public/            все .php страницы и ассеты
public/assets/     стили
data/database.db   локальная SQLite база данных
server.js          Node-скрипт, запускающий PHP сервер
```

## Проверка

1. Зарегистрируйте аккаунт на `/register.php`.  
2. Войдите на `/index.php`.  
3. После входа попадёте на `/portfolio.php`.  
4. Нажмите «Выйти», чтобы вернуться на страницу входа.  
5. Попробуйте сбросить пароль на `/reset_password.php`.
