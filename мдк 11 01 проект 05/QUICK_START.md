# Быстрый старт 🚀

## Минимальные шаги для запуска на новом ПК:

### 1️⃣ Установите ПО (если ещё не установлено)
- **PHP 8.0+** (проверка: `php -v`)
- **Composer** (проверка: `composer --version`)
- **MySQL** (через XAMPP/OpenServer или отдельно)

### 2️⃣ Откройте проект в командной строке
```bash
cd "мдк 11 01 проект 05"
```

### 3️⃣ Установите зависимости
```bash
composer install
```

### 4️⃣ Создайте базу данных
Откройте MySQL и выполните:
```sql
CREATE DATABASE auth_project_test CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

Затем импортируйте схему:
```bash
mysql -u root -p auth_project_test < tests/database/schema.sql
```

**Или через phpMyAdmin:**
- Создайте БД `auth_project_test`
- Импортируйте файл `tests/database/schema.sql`

### 5️⃣ Настройте подключение к БД (если нужно)
Откройте `phpunit.xml` и измените пароль MySQL (если он не пустой):
```xml
<env name="DB_PASSWORD" value="ваш_пароль"/>
```

### 6️⃣ Запустите тесты

**Вариант А: Через командную строку**
```bash
composer test
```

**Вариант Б: Через веб-интерфейс**
1. Запустите PHP сервер:
   ```bash
   php -S localhost:8000
   ```
2. Откройте в браузере:
   ```
   http://localhost:8000/test_results.php
   ```
3. Нажмите кнопку "Запустить тесты"
