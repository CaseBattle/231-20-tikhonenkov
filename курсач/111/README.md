## Проект 06 — адаптивный сайт (МДК 11 01)

Адаптивный одностраничный сайт по макету Figma с Node.js сервером на Express.

Стек:
- Node.js, Express
- HTML5, CSS3 (mobile-first, медиазапросы)
- JavaScript (навигация, бургер-меню, слайдер)

### Установка

```bash
cd "мдк 11 01 проект 06"
npm install
```

### Запуск

```bash
npm start
```

Сайт будет доступен по адресу `http://localhost:3000`.

### Структура проекта

- `server.js` — Node.js сервер на Express, раздающий статические файлы из папки `public`.
- `public/index.html` — главная: градиентный фон, поиск, кнопки «Регистрация/Вход», hero, 2 контентных блока, слайдер, футер с контактами.
- `public/register.html` — регистрация; `public/login.html` — вход.
- `public/admin-login.html` — вход администратора.
- `public/cabinet.html` — личный кабинет пользователя (пример).
- `public/admin-cabinet.html` — личный кабинет администратора (пример).
- `public/styles.css` — стили: зелёный градиент, скругления, мобильное меню, слайдер, медиазапросы.
- `public/script.js` — плавный скролл, бургер, фиксированное меню со сменой стиля при скролле, слайдер с автопрокруткой/пауза/свайпы, динамический год.
- `public/api.php` — заглушка под серверную логику (структура с PHP по ТЗ).
- `public/img/` — PNG‑изображения (замените на экспорт из Figma).

### Что осталось сделать под ваш конкретный макет Figma

- Заменить плейсхолдеры изображений в `public/img` на реальные PNG, экспортированные из макета.
- При необходимости адаптировать текст, цвета и отступы под точные значения из Figma.
